# Frontend boilerplate
> Custom boilerplate for simple html-projects with css-styling and JS

## Stack
- Pug — HTML templating
- postCSS — extending CSS
- ESlint — JS linting
- Stylelint — CSS linting
- Webpack — bundling assets
